# `TextProcessingEvent`

## Properties

### `element`

A survey element (question, panel, page, or survey) for which the event is raised.

**Type**: `Question | PanelModel | SurveyModel | PageModel`

### `name`

The name of the property that contains the text to process.

**Type**: `string`
