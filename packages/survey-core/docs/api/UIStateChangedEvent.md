# `UIStateChangedEvent`

## Properties

### `element`

A survey element whose state change triggered the event.

**Type**: `ISurveyElement`

### `changedProperty`

The name of the UI state property that changed.

**Type**: `"collapsed" | "activeElementName" | "activePanelIndex" | "passed"`
