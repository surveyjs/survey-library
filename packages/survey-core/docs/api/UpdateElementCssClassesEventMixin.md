# `UpdateElementCssClassesEventMixin`

## Properties

### `cssClasses`

An object with CSS classes applied to the element being rendered, for example, `{ root: "class1", button: "class2" }`. You can modify this object to apply custom CSS classes.

**Type**: `any`
